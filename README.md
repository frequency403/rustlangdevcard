# RUSTLANGDEVCARD

A simple terminal-candy ID-Card generator in pure rust

## Installation
### Build from Source by Cargo

```bash
cargo build
```

## Usage

```bash
#Windows
./rustlangdevcard.exe
#Unix
./rustlangdevcard
```
